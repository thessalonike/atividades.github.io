Estudando HTML

Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso porque a linguagem HTML é responsável apenas pela marcação dos elementos na janela do navegador. O HTML não cria nada, ele manda buscar ou ultiliza as letrinhas que você digita.

Parágrafos

Para colocarmos parágrafos, usamos elementos "p" e "/p". para fazer uma quebra de linha, mantendo o mesmo paragráfo, usamos "br". Temos ainda paragráfos especiais, denominados títulos, cabeçalhos ou headings em que são usados elementos de "h1" a "h6" e seus respectivos fechamentos "/h1" a "/h6"

Imagens

Uma imagem aparece na janela do navegador pelo uso do elemento "img". Esse elemento possui um atributo "src", responsável pela localização da imagem. para trabalhar com a imagem, você pode colocá-la dentro das tags de marcação de paragráfo.
