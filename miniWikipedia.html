PHP

PHP (um acrônimo recursivo para "PHP: Hypertext Preprocessor", originalmente Personal Home Page) é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.[3] Figura entre as primeiras linguagens passíveis de inserção em documentos HTML, dispensando em muitos casos o uso de arquivos externos para eventuais processamentos de dados. O código é interpretado no lado do servidor pelo módulo PHP, que também gera a página web a ser visualizada no lado do cliente.

Principais Caracteristicas

A linguagem PHP é uma linguagem de programação de domínio específico, ou seja, seu escopo se estende a um campo de atuação que é o desenvolvimento web, embora tenha variantes como o PHP-GTK. Seu propósito principal é de implementar soluções web velozes, simples e eficientes[23]. Características:

  • velocidade
  • Orientação a objetos
  • Portabilidade - independência de plataforma - escreva uma vez, rode em qualquer lugar.
  • Tipagem dinâmica.
  • Sintaxe similar a C/C++ e o Perl.
  • Open-source
  • Server-side (O cliente manda o pedido e o servidor responde em página HTML)

5 curiosidades sobre o PHP

  1. Uma linguagem para desenvolvimento Web
    Desde o princípio, o PHP foi desenhado como uma linguagem de script para ser usada em servidores, ou seja, foi feita com o desenvolvimento web em mente. O código PHP pode ser incluído diretamente junto como código HTML ou combinado vários frameworks existentes.

  2. Uma linguagem livre
    O PHP é uma linguagem livre e de código aberto, ou seja, você pode usá-la gratuitamente e ainda consultar e até contribuir para o código se assim desejar.

  3. Utilizado por várias empresas grandes
    Você vai encontrar esta tecnologia em vários sites, dentre eles: Facebook, Wikipedia, Tumblr, Slack

  4. Suporte a orientação a objetos
    Desde o PHP 5, a linguagem suporta orientação a objetos que é um paradigma que te permite ter um código muito mais legível, reutilizável e fácil de manter. Este paradigma tem ampla adoção entre as grandes empresas de software e permite organizar seu código em classes e objetos, evitando assim bugs pois te permite isolar cada parte do código no seu lugar e interagindo só com o que você quer.

  5. O PHP tem um mascote GRANDÃO
    É uma tradição muito comum usar mascotes para representar projetos de software livre O PHP não ficou de fora e decidiram fazer seu próprio mascote, o ElePHPant!
